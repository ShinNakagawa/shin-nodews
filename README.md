# Node.js Websocket Test

A tiny demo using the [einaros/ws](http://einaros.github.io/ws/) WebSockets implementation.

# Running Locally

``` bash
npm install
npm start
```

Start the app
You can now start and test the app locally at http://localhost:5000:

# Running on Heroku

``` bash
heroku create
git push heroku master
heroku open
```
